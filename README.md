# AlThabetLanding Component

Landing page component for "Al-Thabet Architectural Decoration", featuring a modern, Arabic-centric design with scroll animations and responsive layout.

## Dependencies
- react
- framer-motion
- lucide-react
- tailwindcss

## Props
This component does not accept any props currently. It is a full-page layout component.

## Usage

```tsx
import { AlThabetLanding } from '@/sd-components/8aa6a93f-8e61-4990-8965-c94ccf1e7f8b';

function App() {
  return (
    <div className="w-full min-h-screen" dir="rtl">
      <AlThabetLanding />
    </div>
  );
}
```
